IT INF 5.1 — Software License Keys

Mandatory
Top ↑

Requirement / 原文要求

Virtual infrastructure Boehringer Ingelheim is purchasing and setting up required hardware servers for the virtual infrastructure. The vendor shall not deliver any hardware servers. The virtual infrastructure (VMWare ESXi host, vCenter) is set up by the Boehringer Ingelheim IT as grouped clusters. The following features are active and must be supported by the vendor: vSphere High Availability (HA): in case of a ESXi host failure, the effected Virtual Machines (VMs) will automatically be restarted on the other hosts within the cluster Load balancing Distributed Resource Scheduler (DRS): The load of the VMs is balanced between the hosts within the cluster, vMotion is used to move VMs between the ESXi hosts. vSAN: The direct attached local disks (no RAID controller) of each host managed as cluster are presented as a shared storage . Boehringer Ingelheim is purchasing and managing required licenses for the virtual infrastructure. Boehringer Ingelheim is patching virtual infrastructure when patches are available.

中文解读 / Chinese interpretation

BI 采购并搭建虚拟化基础设施所需的硬件服务器，供应商不得交付任何硬件服务器。虚拟化基础设施（VMware ESXi Host、vCenter）由 BI IT 建立为分组集群，已激活 vSphere HA（主机故障时 VM 自动在其他主机重启）、负载均衡 DRS（VM 负载在集群间均衡，vMotion 迁移 VM）和 vSAN（本地磁盘作为集群共享存储）。BI 负责采购和管理虚拟化许可及补丁。此要求用于确保供应商应用可在标准 BI 虚拟化平台上运行。

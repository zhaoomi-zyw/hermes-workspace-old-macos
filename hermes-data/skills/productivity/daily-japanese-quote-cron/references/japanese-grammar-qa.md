## ら抜き言葉 (ra-dropping) — 食べれる vs 食べられる

**Core rule:** For ichidan verbs (一段動詞), the standard potential form is `~られる`, but colloquially the `ら` is often dropped → `~れる`. This is called 「ら抜き言葉」.

| Form | Meaning Scope |
|------|--------------|
| 食べ**られる** | 可能・受身・尊敬 (potential / passive / honorific — 3 meanings, ambiguous) |
| 食べ**れる** | 可能のみ (potential ONLY, no ambiguity) |

### Why it evolved
`食べられる` alone carries potential, passive, AND honorific meanings simultaneously, forcing context disambiguation. Native speakers naturally dropped `ら` in potential usage to resolve ambiguity.

### When to use which
- **Casual / spoken:** 食べれる is totally fine — 99% of Japanese use it in conversation
- **Formal / written / JLPT exams:** Must use 食べられる — ら抜き will be marked wrong

### Other ichidan verb examples

| Verb | Standard Potential | ら抜き (colloquial) |
|------|-------------------|---------------------|
| 見る | 見られる | 見れる |
| 起きる | 起きられる | 起きれる |
| 寝る | 寝られる | 寝れる |
| 出る | 出られる | 出れる |
| 着る | 着られる | 着れる |
| 考える | 考えられる | 考えれる |

### Mnemonic
「ら」を抜くと「可能だけ」= Drop the `ra`, keep only "can"

## Vocabulary Notes

### 麻辣烫 (málàtàng)
- Japanese: **マーラータン** (mārātan) — phonetic loanword
- Popular in Tokyo (Ikebukuro, Shinjuku — e.g. 七宝麻辣湯)
- Spiciness levels: なし (none) / 微辛 (mild) / 普通 (medium) / 激辛 (super hot)

### 万能呼びかけ: すみません
Single most versatile word in Japanese service contexts:
- Calling a waiter → すみません
- Apologizing → すみません
- Getting attention → すみません
- Asking for help → すみません

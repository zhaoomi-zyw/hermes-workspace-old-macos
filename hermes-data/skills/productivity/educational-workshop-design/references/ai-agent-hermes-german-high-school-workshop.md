# AI Agent / Hermes Workshop for German High School Intern

## Context

The user is designing a 2-hour workshop for a German high school student who studies in Germany and is doing a short internship in China.

The workshop topic is the shift from traditional chatbots such as ChatGPT to AI agents such as Hermes.

The planning conversation may happen in Chinese, but final learner-facing outputs should be in English.

## Confirmed Requirements

- Audience: German high school student, likely beginner-friendly level.
- Location/context: short internship in China.
- Total duration: 2 hours.
- Student's specific interests are not known in advance.
- Start with an interest-discovery / icebreaker segment.
- Overall style: balanced.
  - Inspire curiosity.
  - Provide hands-on experience.
  - End with future learning / career relevance.
- Final student takeaways:
  - One small HTML artifact created during the workshop.
  - A reusable set of AI agent prompt templates.
- Discussion with the user can be in Chinese.
- Final HTML courseware, outline, slide copy, prompt templates, and learner-facing materials should be in English.

## Recommended Workshop Spine

1. Interest discovery — 10 min
2. Chatbot vs Agent concept — 20 min
3. Hermes demo: idea to HTML artifact — 25 min
4. Agent as study/research assistant — 20 min
5. Student hands-on challenge — 25 min
6. Future of AI agents and careers — 10 min
7. Setup and next steps — 10 min

## Interest-to-Demo Mapping

Because the student's interests are unknown, prepare adaptable demo routes:

- Coding / AI / technology → mini website or browser game
- School / exams → AI study coach
- China internship → China internship diary or survival assistant
- Football / sports → match or player analysis assistant
- Gaming / esports → gaming stats or strategy assistant
- Future career → AI career exploration plan

## Core Explanation

Simple framing:

- A chatbot answers questions.
- An agent completes tasks.

Analogy:

- ChatGPT is like a smart person in a chat window.
- An AI agent is like a smart assistant sitting at a computer.

Memorable lines:

- ChatGPT is a brain that can talk; an agent is a brain that can use tools.
- The old skill was searching the web.
- The current skill is prompting AI.
- The next skill is directing agents.

## Suggested Student Takeaway Prompt Templates

Include copy-paste-ready templates such as:

1. Study Coach
2. Research Assistant
3. Career Explorer
4. China Internship Diary Helper
5. Coding Buddy
6. Website/Game Builder

## Delivery Notes

For the first experience, prefer using the already configured Hermes WebUI on the instructor's machine rather than making installation the first task.

If the student wants to continue later, recommend a staged path:

1. Use WebUI or a hosted/simple interface first.
2. Install VS Code.
3. Install Hermes CLI.
4. Later learn skills, memory, automation, GitHub, and deployment.

Avoid introducing gateway, profiles, MCP, and advanced provider configuration too early.

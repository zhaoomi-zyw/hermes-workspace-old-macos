# 宠物行业竞品知识库结构

> 模板：专业级竞品情报 Obsidian 知识库组织规范
> 适用场景：BI 中国宠物团队，宠物驱虫药竞品分析

## 目录结构

```
vault-root/
├── .obsidian/                # Obsidian 配置
├── 00-MOC/                   # 地图索引（Map of Content）
│   └── 主索引.md             # 总入口，链接所有子目录
├── 01-竞品公司/              # 每家公司一个文件
│   └── 公司名.md             # 基本信息+产品线+竞争分析+[[相关笔记]]
├── 02-产品清单/              # 产品对比矩阵
│   ├── 产品清单.md           # 全量表
│   └── 产品属性矩阵.md       # 多维度交叉分析
├── 03-市场数据/              # 宏观数据
│   ├── 市场规模.md
│   └── 趋势报告.md
├── 04-财报分析/              # 财务数据
│   ├── 季度业绩汇总.md
│   └── 数据勘误记录.md
├── 05-周报/                  # 日/周报存档
├── 06-战略分析/              # 深度策略报告
└── 99-附件/                  # PPTX、PDF 等非 Markdown 文件
```

## 命名规范

- **目录**：数字前缀保证排序，中文描述性名称
- **公司文件**：`英文名-中文名.md`（如 `BI-勃林格殷格翰.md`）
- **产品清单**：描述性中文名
- **周报**：`YYYY-WXX周-YYYYMMDD.md`（如 `2026-W23周-20260605.md`）
- **财报**：`YYYY-QX-描述.md`（如 `2026-Q1-全球宠物动保业绩.md`）

## 笔记模板

### 公司档案
```markdown
---
tags: [公司, 跨国药企/国产, 竞品]
aliases: [简称1, 英文名]
---

# 公司全称-中文名

## 基本信息
- 全称/总部/定位
- 关键财务数据

## 产品线（驱虫药）
| 产品 | 剂型 | 覆盖 | 备注 |

## 竞争分析
- 优势/劣势/威胁

## 相关笔记
- [[链接其他公司]]
- [[链接产品清单]]
- [[链接财报]]
```

### 主索引页
所有的 `[[wikilink]]` 指向具体笔记，按目录分组展示，底部放「常用快捷链接」。

## 关键约定

1. **所有笔记间用 `[[ ]]` 双向链接**，不要纯文本
2. **YAML frontmatter** 必须包含 `tags` 和 `aliases`
3. **数据可信度标注**：✅ 公开验证 / ⚠️ 内部估算 / ⚠️ 无法验证
4. **勘误单独成文**，不混入正文
5. **保留原有文件**，不随意删除

# Guided Agent Workshop Prompt Pack Pattern

Use this reference when designing a beginner workshop where the learner should operate an AI agent without having to invent prompts live.

## When to Use

- The learner is a student, intern, beginner, or non-specialist.
- The instructor wants the learner to personally paste prompts and see the agent work.
- The session has a fixed duration and must avoid open-ended prompt improvisation.
- The workshop should produce a visible artifact plus reusable assistant patterns.

## Design Principles

1. Prepare prompts before the session.
2. Make prompts copy-paste ready, with no hidden placeholders unless clearly marked.
3. Each prompt should produce a predictable outcome.
4. Include refinement prompts, not only first-shot generation prompts.
5. Let the learner choose from a small set of paths, but keep a default fallback.
6. Keep the instructor responsible for explanation, branching, and recovery.
7. Separate classroom prompts from post-class reusable prompts.

## Recommended Prompt Pack Structure

### 1. Interest Discovery Prompt

Purpose: convert the learner's hobbies and goals into a concrete mini project.

Include:
- 5 short questions
- suggested project paths
- a recommendation based on the answers

### 2. Default Artifact Creation Prompt

Purpose: create the visible classroom artifact.

Good defaults:
- personal mini website
- internship diary page
- AI learning portfolio
- simple browser game
- study coach dashboard mockup

For international students in an internship context, a strong fallback is:
"My China Internship Diary".

### 3. Refinement Prompts

Purpose: teach that agent work is iterative.

Include prompts for:
- make it more professional
- adjust colors and layout
- add a section
- simplify the text
- make it more personal
- make it mobile friendly

### 4. Personal Assistant Prompts

Purpose: give the learner useful patterns after the class.

Common templates:
- study coach
- research assistant
- career explorer
- internship diary helper
- coding buddy
- language/culture helper

### 5. Safety / Scope Prompt

Purpose: teach responsible agent use.

Include reminders:
- do not paste private passwords, API keys, or sensitive school/company information
- verify facts before using them in important work
- ask the agent to explain what it changed
- start with small tasks before giving broad permissions

## Prompt Quality Checklist

Before finalizing a prompt pack:

- The learner can paste each prompt without editing it.
- The expected output is clear.
- The prompt constrains the artifact enough to avoid drift.
- The prompt asks for real files or visible results when appropriate.
- There is a follow-up prompt for improvement.
- The language matches the student-facing language requirement.
- The default path works even if the learner has no strong stated interests.

## Example Default Flow

1. Learner pastes interest-discovery prompt.
2. Instructor chooses a route from the answer.
3. Learner pastes artifact-creation prompt.
4. Agent creates first version.
5. Learner pastes one refinement prompt.
6. Instructor explains the difference between asking a chatbot and directing an agent.
7. Learner leaves with the artifact and a prompt handout.

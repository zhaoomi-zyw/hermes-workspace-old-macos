Instructor Runbook

A 2-hour operation script. Student-facing pages are trilingual; this page can include Chinese instructor notes.

0:00-0:10 Interest discovery

Open index.html. Ask: “If you had an AI assistant that could use this MacBook, what would you want it to do?”

中文备注：根据回答选择路线；没有明确兴趣就走 China Internship Diary。

0:10-0:30 Chatbot vs Agent

Open chatbot-vs-agent.html. Emphasize: chatbot answers, agent completes tasks.

中文备注：不要讲太多架构，先让他建立直觉。

0:30-0:55 Demo 1

Open live-demo-guide.html. Let the student paste Prompt 1 and Prompt 2.

If generation fails or is slow, open fallbacks/my-china-internship-diary.html.

0:55-1:15 Demo 2

Open agent-prompt-templates.html. Use Study Coach or Career Explorer prompt.

1:15-1:40 Hands-on challenge

Student chooses a route. Keep it bounded. Use prepared prompts only.

1:40-1:50 Future trend

Say: “Search literacy was important in the 2000s. Prompt literacy is important now. Agent literacy will be important next.”

1:50-2:00 Clean MacBook / next steps

Open ways-to-use-hermes.html and student-next-steps.html. Switch to clean MacBook for install demo if time allows.

中文备注：如果安装耗时，展示关键命令和说明课后继续完成。

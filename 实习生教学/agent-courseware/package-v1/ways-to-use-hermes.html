Ways to Use Hermes

Which way should a beginner start with?

WebUI

Recommended first. Chat-like, visual, easy for copy-paste prompts.

Use Omi's current third-party nesquena/hermes-webui setup, not the built-in dashboard.

CLI

Great for technical learners and coding. Show as the next step, not the first hurdle.

Messaging / Gateway

Powerful for daily mobile use, but too much setup for the first student session.

Cron / Automation

Useful for reminders and scheduled study tasks after the student understands agents.

Profiles / MCP

Advanced. Mention as deeper capability, do not teach in detail.

Clean MacBook path

Install the minimal route first, then open WebUI or show CLI as advanced.

Recommendation

Start with WebUI on the configured MacBook. Then demonstrate the clean MacBook installation path. Show CLI only as the advanced next step.
Starte mit der WebUI auf dem bereits eingerichteten MacBook. Danach zeige die Installation auf dem sauberen MacBook. CLI nur als fortgeschrittenen nächsten Schritt zeigen.
主体验用已配置机器的 WebUI；再用 clean Mac 演示安装；CLI 作为进阶展示。

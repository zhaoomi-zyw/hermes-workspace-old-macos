Prompt 1: create the artifact

Create a single-file HTML page called "My China Internship Diary" for a high school student doing a short internship in China. Use English as the main language, and include short German translations under important sections. The German translations should be brief and supportive, not full duplicate paragraphs. Make the design modern, clean, and professional, not childish. Use a reflective but natural tone, suitable for a high school student, but keep the page visually professional. Before generating the final HTML file, ask me several guided questions about my internship experience in China. Wait for my answers, then use them to enrich and personalize the content of the HTML page. The questions should cover topics such as: - What was my first impression of China? - What did I do during the internship? - What did I learn from the workplace or team? - What cultural differences or observations did I notice? - Which AI tools did I try, and how did I use them? - What surprised me most? - What questions do I still have after the internship? The final HTML page should include these sections: - First impression - What I learned - Cultural observations - AI tools I tried - Next questions Save the result as an HTML file and tell me the exact file path.

Prompt 2: refine the artifact

Please improve the HTML page visually. Keep it readable and professional. Add a clear hero section, cards for each internship insight, and a small "What I want to learn next" checklist. Do not make it flashy or childish. Keep English and German visible. The final output should be an HTML-based PPT-style slide deck, with clear slide sections that can be presented like a short presentation.

If live generation fails

Open fallbacks/my-china-internship-diary.html and continue the explanation from there.

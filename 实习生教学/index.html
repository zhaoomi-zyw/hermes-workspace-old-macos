2-hour beginner workshop

From Chatbots to AI Agents

How tools like Hermes turn AI from a chat window into a real digital assistant that can use files, code, browser tools, memory, and automation.

Workshop goals

1

Understand the shift

A chatbot answers questions. An agent completes tasks.
Ein Chatbot beantwortet Fragen. Ein Agent erledigt Aufgaben.
聊天机器人回答问题；Agent 完成任务。
2

See Hermes in action

Hermes can create files, edit code, search, remember context, and automate work.
Hermes kann Dateien erstellen, Code bearbeiten, suchen, Kontext merken und Arbeit automatisieren.
展示 Hermes 可操作文件、代码、搜索、记忆和自动化。
3

Leave with something useful

You will create a small HTML artifact and keep reusable agent prompts.
Du erstellst ein kleines HTML-Projekt und bekommst wiederverwendbare Agent-Prompts.
学生带走 HTML 小作品和可复用 prompt。

一、第一册（初级）

第十四课
语法用法接续例句备注
に基准N/A 一週間に三回ジムに行きます。
で原因N/A お腹が空いたのでたくさんご飯を食べました。 遠くで、新幹線で行きます。 病気で家で休みます 很容易变成「ので」 感觉不常用
ようだ比喻・比况・推测 どうも王さんが日本人のようです
ても（でも）いい可以，…也没关系 勉強できなくても大丈夫ですか
も 肯定：数量之多
否定：数量之少		 今日は日本語10時間も勉強します。 昨日は30分も走りませんでした 今天竟然要学习日语10小时
でも举例 喉が渇いたので、何か飲み物でもありますか
〜と〜と、どちらが〜ですか
〜のほうが〜		 两个或以上事物的比较 コーヒーと抹茶と、どちらがいいですか。 コーヒーのほうがいいです
第十五课
语法含义接续例句备注
ために目的，为了… 名词 + の + ために 动词原型 + ために 日本に仕事するために、日本語を頑張ります。 生活のために、毎日働いています。
できる可能性・能力・许可 1. 动作性名词 + できる 2. 动词原型 + ことができる 3. 名词 + ができる このお手洗いが利用できます この薬が飲むことができる スキーができる
て（で）も即使 一类形容词 → 词干 + くて 二类形容词 → 词干 + で 名词 → で 动词 → て形 たとえ雪が降っても、行きたいと思います

二、第二册

第1课
语法含义接续例句备注
てから先…后… 动词て形 + から 仕事を終わってからビールを飲みました
〜しか〜ない只有…、只… 动词ない形 私はコーヒーしか飲まない 私はコーヒーしか飲みません 对比：私はパンだけ食べる 私はパンしか食べません
〜ようになる变得…（从不可能到可能） 基本形 + ようになる 否定：ない形 + なくなる 彼は日本語が上手になるようになりました 学校に行かなくなりました
〜（の）ほかに除…之外 基本形 + ほかに 名词 + の + ほかに 日本のほかに、どこにも行きません。 お酒を飲むほかに、タバコを吸います
〜ていく 空间：由近及远（做完去）
时间：从现在持续到未来		 て形 + いく コーヒーを飲んで行ってください。 天気が暑くていく
〜つもり打算… 动词原型 + つもり 寿司を食べるつもりです ないつもり：不打算（语气温和） つもりはありません：不打算（语气强烈） つもりではありません：澄清辩解
动词意志形决定要做什么 サ変：掃除する → 掃除しよう カ変：来る → 来よう 一段：食べる → 食べよう 五段：買う→買おう、書く→書こう 遊ぶ→遊ぼう、休む→休もう
意志形 + と思う／思っている 我打算… 意志形 + と思う（我想） 意志形 + 思っている（别人想 / 我一直想） 日本で焼鳥を食べようと思います 日本で焼鳥を食べようと思っています 王さんは日本で焼鳥を食べようと思っています。

语法含义接续例句备注
形容词性动词
（優れる・富む・溢れる）

第3课 — 自动词・他动词
语法含义接续例句备注
自动词・他动词 自动词：现象/状态（着）
他动词：把/让…（动作）		 名詞が…（自动）
名詞を…（他动）		 ドアが開く（自动） ドアを開ける（他动）
ている 1. 正在做某事
2. 状态
3. 习惯		 动词 + ている 1. 王さんはご飯を食べている 2. ドアが開いている 3. 毎日運動している
てある 有意图的结果（人为形成） …が + 他动词 + てある 換気のために、ドアが開けてあります 强调目的性：门被打开是有原因的
さ事物的程度 形容词词干 + さ 高い → 高さ　富士山の高さは日本で一番です 静か → 静かさ　暑い → 暑さ
にする选择、决定 选的东西 + にする ケーキにする／ビールにする お昼は何にしますか → ラーメンにする どちらにしますか
そうだ 视觉直觉判断（样态）
可能性很大（好像要）		 形容词：去い + そう 形容动词：直接 + そう 动词：去ます + そう 雨が降りそうです このケーキは美味しそうです 否定：降りそうにない 降りそうもない 降りそうにもない
みたいだ 「ようだ」的口语
1. 推测
2. 像什么一样		 前接续：名词词干／形容动词词干 形容词简体／动词简体 后接续：な + 名词／に + 句子 です结句（多用于推测） 子どもみたいだね 静かみたいだ／寒いみたいだ 雨が降るみたいだ 和「ようだ」的前接续区别： ようだ → 名词 + の、形容动词 + な
第4课
语法含义接续例句备注
に作为 名词 + に プレゼントに花をあげる
〜によると〜そうだ听说、据说（传闻） 简体句 + そうだ 新聞によると、明日は台風が来るそうです この店のラーメンはおいしいそうだ。 彼は元気だそうだ。／あの人は先生だそうだ。 注意与样态「そうだ」的区别： によると／王さんの話では + 句子 + そうだ
そうだ（样态）好像（眼前判断） 一类形容词去い + そう 二类形容词去だ + そう 动词去ます + そう このケーキはおいしそうです この部屋は静かそうです 雨が降りそうです
ちょうど〜ているところだ 现在：正在…的时候
过去：刚刚结束		 動詞ている + ところです 動詞た形 + ところです ちょうど走っているところです 宿題を終わったところです 食べるところ（马上要吃） 食べたところ（刚吃完） 食べているところ（正在吃）
〜たがる想…（第三人称） 动词ます形去ます + たがる 王さんは走りたがっています 子供は遊びたがっています 注意：たがらない 食べたいと言っています
〜たい想…（第一人称） 动词ます形去ます + たい この映画をみたいね
〜のに目的：为了、对于 动词原型 + のに ゲームをするのに、PS5を買いました 和「ために」基本用法一致 另有「明明…却」的意思
第5课 — 可能态
语法含义接续例句备注
〜とは〜（こと）です
〜というのは〜（こと）です		 下定义：所谓…就是… 简体句子 + ということです 名词 + のことです 才能とは、自分を信じることです。 留学とは、外国へ行って、そこで勉強するということです
可能态（れる・られる）能、能够、可能性 一段动词：去る + られる サ変：去する + できる カ変：くる → こられる 五段：末假名変え段 + る 食べられる／掃除できる／こられる 書ける、使える、見える、飲める ⚠️ 无可能态的动词： 存在动词、見える／聞こえる／分かる 咲く、晴れる、降る ⚠️ 可能态后全部变自动词（が）
見える vs 見られる
聞こえる vs 聞ける		 見える／聞こえる：自动映入
見られる／聞ける：需条件才能		 天気がいいから、月食が見られる 猫は夜も目が見える お腹が悪いから、食べられません。
〜てくる时间空间由远及近 ビールを買ってくる（买酒来） ちょっと手紙を出してくる（我去寄个信就来）
〜ていく时间空间由近及远 お弁当を使って行きます。（做了便当就去） 薬を買っていきます。（买了药再去）
〜まで直到…为止 病気が治るまで、家で休みます
〜てくださいませんか请…好吗（尊敬） 窓を開けてくださいませんか
少しも〜ない一点也不… 少しも暑くない 英語が少しもわからない。
なんとか好歹、总算 なんとか間に合った。（总算赶上了） 彼はなんとか試験に合格した。 「なんとかなるよ」= 总会有办法的
第6课 — 授受关系（恩惠的转移）
动词主语对象例句
あげる／さしあげる／やる我对方 我给小李钱 → 私は李さんにお金をあげる
くれる／くださる对方我 小李给我钱 → 李さんは私にお金をくれる
もらう／いただく我对方 我收到小李的钱 → 私は李さんに（から）お金をもらう

语法含义接续例句备注
〜てばかりいる老是…、总是…（负面） 动词て形 + ばかりいる テレビを見てばかりいる。 → 总是看电视（暗示不学习）
がほしい想要某物 名词 + がほしい お茶がほしいです
〜てほしい希望别人做某事 动词て形 + ほしい 早く帰ってほしい。 もっと勉強してほしい。 静かにしてほしい。 主语通常是"我"，动作执行者是对方
たい／たがる想要做… ます形去ます + たい（第一人称） ます形去ます + たがる（第三人称） 休みたい 李さんは休みたがる
なければいけない必须… 动词ない形 + ければいけない 行かなければいけない（必须去） 食べなければいけない（必须吃） 勉強しなければいけない（必须学习）
第7课 — 授受表达 + 接续
语法含义接续例句备注
〜てあげる我给别人做… AはBに〜てあげる／さしあげる／やる 私はお母さんに掃除してあげました
〜てくれる别人给我做… AはBに〜てくれる／くださる 王さんは私に授業を教えてくれました てくれますか = 你为我做…吗？
〜てもらう我收到别人做…的恩惠 AはBに〜てもらう／いただく 王さんに授業を教えてもらいました てもらえますか = （我能得到）你为我做…吗？
ないで没…就… ないで + 动词 学校に行かないで、ずっと家で寝ています
までに动作的期限（一次性） 試験は九時までに始まります
まで持续到…（持续） 試験は九時まで続きました
のに明明…却… 动词／形容词简体 + のに 名词／形容动词词干 + な + のに 雨が降っているのに出かけました
それともA还是B（选择） コーヒーにしますか？それとも紅茶がいいですか
なくて vs ないで なくて：因果
ないで：状态下做某事
第8课 — 被动态
语法含义接续例句备注
〜ことがある有时候会、偶尔 动词原型 + ことがある 食べることがある
〜たことがある有某种经历 动词た形 + ことがある 食べたことがある
によって（被动句） 由…（创造性的对象）
書く、作る、建てる、発明する、設計する		 …は…によって〜られる この本は私によって書かれました この建物は有名な建築家によって設計されました
被动省略主语 活动举行一般用被动 コンサートが公園で開かれました
持ち主の受身 我的…被别人… 私は〜に〜を〜 王さんは私に足を踏まれた。（我的脚被小王踩了）
迷惑の受身 受到动作造成的影响 私は子供に泣かれた。（小孩哭，我被影响） 家の前に高いビルを建てられた。 雨が降られた。
〜てもらいたい 想让别人为自己做… 明日来てもらいたい（我想你明天来）
かどうか 是不是…、是否… 动词／形容词简体 + かどうか 名词／形容动词词干 + かどうか 学生かどうか 王さんが日本へ行くかどうか
第9课 — 目的・样态
语法含义接续例句备注
〜ように ① 目的：为了…、为了使得…
② 看起来好像（实际不是）
③ 像…一样		 ① 动词简体／可能态 + ように
② 普通形 + ように見える
③ 名词 + の + ように
   动词简体 + ように		 ① 遅れないように早く家を出ました 日本語が話せるように毎日練習しています ② 彼女は冷たいように見えるが、本当は優しい人だ ③ 鳥のように飛ぶ／鳥が飛ぶように走った
〜ために为了…（明确目的） 动词简体 + ために 試験に合格するためにメモする
〜たまま保持原样不变 动词た形 + まま テレビをつけたまま寝てしまった
〜ように言う转述命令/请求/劝告 动词普通形 + ように言う 母は私に野菜を食べるように言った
〜ておく事先准备 动词て形 + おく 会議の資料をコピーしておいてください
〜なんか列举（类似「など」） 名词 + なんか 寿司なんかどう？
第10课 — 条件・假定
语法含义接续例句备注
〜として作为… 名词 + として 山田さんは詩人として有名です
〜と一…就… 动词普通形 + と + 结果 春になると、桜が咲きます
〜たばかり刚刚… 动词た形 + ばかり 日本に来たばかりの頃は、何もわかりませんでした
別に〜ない并不特别…、没什么… 別に + 否定形式 別に困っていない
いくら〜ても无论怎么…也… いくら + 动词／形容词て形 + も いくら勉強しても、漢字が覚えられない
〜ば 假如…、如果… 五段動詞：え段 + ば い形容詞：去い + ければ な形容詞／名詞：なら（ば） 飲めれば、乾杯しましょう。 忙しければ、無理しないでください。
疑问词 + 动词か 是否…／什么…（间接疑问） 疑问词 + 简体句 + か 彼がどこに行ったか知りません。 何を食べたいか教えてください。 彼女がいつ来るかはまだ決まっていない。 どうすればいいか分からない。
第11课 — 命令・禁止・推测
语法含义接续例句备注
〜ぬ否定（古语/惯用） ます形去ます + ぬ 言わぬが花（不说才是美） 知らぬが仏（不知者为佛）
〜なさい必须…（上对下，比てください严肃） ます形去ます + なさい 早く寝なさい
〜ないでいる故意不做… 动词ない形 + ている 言わないでいる（故意不说） その問題を考えないでいると、解決できない ⚠️ 今年は雪が降らないでいる 是不对的
动词命令形 1. 命令（上对下）
2. 提醒或引起注意		 出来ているかどうか、よく確かめろ。 早く寝ろ！
への给…的… 名词 + への + 名词 母への手紙（给妈妈的信）
かもしれない表推测（主观） 名词・形容动词词干 + かもしれない 动词／形容词简体 + かもしれない 彼は病気かもしれない。（他可能生病了→主观） 🆚 推测表达对比： 有客观依据 → らしい／ようだ 寻求认同 → だろう／でしょう 随便说说 → かもしれない 正式 → かもしれません
〜な禁止：请不要… 动词简体 + な 落とすな（禁止扔）
〜てはいけない禁止：不可以… 动词て形 + はいけない ここで食べてはいけません この部屋に入ってはいけない
ほど〜ない A没有B那么… AはBほど〜ない 日本語は中国語ほど難しくない 台北は東京ほど寒くない
第12课 — 条件・建议
语法含义接续例句备注
〜だけでなく〜も不仅…还… 名词・形容词・动词简体 + だけでなく 勉強するだけでなく運動も 肉だけでなく野菜も／静かだけでなく綺麗も 高いだけでなく便利も
〜うちに趁着…的时候
在…期间		 動詞ている + うちに 形容詞词干 + うちに 名詞 + の + うちに 雨が降っているうちに、洗濯物を取り込んだ方がいいよ 空が明るいうちに、写真をたくさん撮っておこう 学生のうちに、いろいろな経験をしておくべきだ
なら ① 既然…那就…（承接） ② 如果是…的话（评价建议） 动词普通形 + なら 形容词词干 + なら 名词 + なら 眠いなら寝ればいい。 ビールなら、生ビール
たら 如果…就…、…之后就… 动词た形 + ら 食べたら、寝る。 父は帰ってきたら、相談しよう 向こうに着いたら、ラインで連絡しよう。
〜まで 甚至连…／竟然到…地步 名词 + まで 彼まで泣いた。／納豆まで好き。
〜といい要是…就好了（愿望） 动词普通形 + といい 明日雨が降らないといいですね。
AかBA或者B 名词A + か + 名词B 電車かバスで行きましょう
〜たら（どうですか）建议：要不要…？ 动词た形 + ら + どうですか そんなに疲れているなら、少し休んだら？
〜た（想起）原来…（突然想起） 动词た形 確か今日は会議があったな（我记得今天有会议来着）
ながら一边…一边… ます形去ます + ながら ラジオを聞きながら、仕事をする

三、第三册（中高级）

第1课
语法含义接续例句备注
ほど程度：达到…的程度 名词／动词简体形 + ほど 忙しくて食事をする時間もないほどだ 字が読めないほど部屋が暗いです 数量程度 → ほど／くらい都能用 比较变化 → 用ほど 轻蔑语气 → 用くらい

② 推进对话（那么…）		 李さんは病気で、それで私は代わりに来ました。 昨日彼と試合をしました。ーそれで、どちらが勝ちましたか？

第4课
语法含义接续例句备注
とは所谓的…（= というのは） 名词 + とは 「友情」とは、互いに信頼し合う関係のことです
において在…（场所/时间/领域）= で 名词 + において 会議において新しい方針が決定された。 〜においても：即使在…也 〜においては：在…方面 〜においての：在…中的
〜を始めとする以…为首／以…为代表 名词 + を始めとする + 名词 サッカーを始めとするスポーツが若者に人気だ
〜にかかわる与…有关／关系到… 名词 + にかかわる 命にかかわる病気（危及生命的疾病） 環境にかかわる問題
〜とともに随着…同时发生 名词 + とともに 社会の発展とともに人々の生活も豊かになった
〜とする／〜とされる 把…当作…／被认为是… 名詞 + とする／とされる 安全を第一とする。（安全作为第一位） この薬は効果が高いとされる。（这种药被认为效果很好）
〜という传闻：据说… 普通形（简体句）+ という 彼は来月結婚するという。 この店のラーメンはとても美味しいという。
〜ものですから因为…所以…（= ので，更口语化） 动词：辞書形／た形／ない形 + ものですから い形：〜い + ものですから な形：〜な + ものですから 名詞：〜な + ものですから 子どもが熱を出したものですから、会社を休みました。常用于解释自己的情况
ですもの 因为嘛…、毕竟… 动词：辞書形／た形／ない形 + ですもの い形：〜い + ですもの な形：〜だ + ですもの 名詞：〜だ + ですもの 寒いですもの、外に出たくない
〜ばかりか 不仅…而且… 普通形 + ばかりか 彼女は日本語ばかりか、中国語も話せる。

# GLP-1 / MASH 中国竞争者生态与市场分析报告

覆盖已上市药品、未来三年上市窗口、Digital Solution 与医保策略  
Prepared for BI Human Pharma Marketing  
数据截至：2026-05-25；公开资料整理，监管审批与商业化时间存在不确定性。

---

## 1. 执行摘要

中国 GLP-1 竞争正在从“谁先上市”转为“谁能把减重、糖尿病、肝脏与长期管理做成闭环”。MASH 短期缺少中国获批标签，但将成为下一轮差异化和医保价值论证的关键战场。

| 维度 | 核心判断 | 市场含义 |
|---|---|---|
| 已上市格局 | MNC 高端心智 + 国产快速追赶 | 诺和诺德、礼来仍掌握品牌与疗效标杆；信达、华东、银诺、先为达等本土企业正在以机制、价格、渠道切入。 |
| 未来三年 | 类似药放量 + 双靶点/口服迭代 | HRS9531、司美格鲁肽类似药、survodutide、口服 GLP-1 与复方组合将改变竞争节奏；上市时间需以 CDE/NMPA 为准。 |
| Digital Solution | 患者旅程闭环成为差异化 | 成熟玩家把药品嵌入小程序、DTP/电商、线上问诊、复购提醒、生活方式管理和数据随访。 |
| 医保策略 | T2D 可进医保；减重和 MASH 更依赖商保/自费 | NRDL 原则仍是“保基本”：T2D 治疗有准入路径，单纯减重支付受限；MASH 需先建立诊断与临床终点价值。 |

### 对 BI 的含义

| 战场 | 竞争焦点 | BI 应对 |
|---|---|---|
| 肥胖/超重 | 减重幅度、耐受性、渠道覆盖、支付能力 | 不只讲体重，绑定肝脂、心代谢和长期依从性。 |
| T2D | 医保准入、医生教育、国产替代价格 | 证据链需聚焦高风险代谢人群，而非泛糖尿病红海。 |
| MASH | 诊断筛查、组织学/非侵入指标、肝纤维化终点 | 提前建设肝病-内分泌-体重管理协同路径。 |

## 2. 中国 GLP-1 / MASH 治疗生态

### GLP-1：从降糖到体重管理

| 场景 | 市场角色 |
|---|---|
| T2D | 医保目录入口、内分泌医生教育、院内处方。 |
| 肥胖/超重 | 自费/商保为主，院外渠道、体重管理中心、线上问诊扩张。 |
| 心肾代谢 | CKD/CV 等适应症扩大生命周期，强化长期治疗价值。 |

### MASH：从肝病诊断到代谢共病管理

| 场景 | 市场角色 |
|---|---|
| 筛查入口 | 肝酶异常、脂肪肝影像、肥胖/T2D 高危人群。 |
| 治疗终点 | MASH 改善、纤维化改善、肝脏事件长期降低。 |
| 商业瓶颈 | 诊断链条长、活检/非侵入检测标准、医保价值尚需论证。 |

**营销启示：** MASH 不是孤立肝病市场，而是 GLP-1 竞争升级后的“高价值共病场景”。

## 5. MASH 市场：已上市与近三年高关注产品

截至本报告日期，未见中国获批 MASH 适应症治疗药物。中国临床治疗仍以减重、控糖、控脂和生活方式为基础。全球层面，MASH 已进入药物时代，resmetirom 与 semaglutide 2.4mg 已形成重要标签信号。

| 产品/企业 | 机制 | 全球状态 | 中国三年窗口判断 |
|---|---|---|---|
| Rezdiffra / Madrigal | THR-β 激动剂 | 美国加速批准；欧盟条件批准 | 若启动中国注册，有望成为 MASH 标签教育锚点；当前未见中国获批。 |
| Wegovy / 诺和诺德 | GLP-1RA 2.4mg | 美国获批 MASH F2-F3 成人适应症 | 中国已有减重标签，MASH 标签扩展需中国注册策略与数据支持。 |
| survodutide / BI-Zealand | GCG/GLP-1 双激动剂 | MASH II 期阳性，计划 III 期 | 最相关的 BI 机会；三年内更现实是建立 KOL、筛查和证据准备。 |
| efruxifermin / Akero 等 | FGF21 类 | 全球 III 期/后期开发 | 中国短期商业化不确定，作为机制竞争监测对象。 |
| 玛仕度肽 / 信达 | GCG/GLP-1 双激动剂 | 中国减重/T2D 已获批 | 可借肝脂改善数据抢占 MASLD/MASH 前置心智。 |

## 6. 竞争者生态图

药品竞争正在被渠道和服务层放大。单纯产品上市不再足够，真正的护城河来自“筛查-处方-支付-复购-随访”的全链路控制。

| 生态角色 | 代表企业/主体 | 影响路径 |
|---|---|---|
| MNC 疗效标杆 | 诺和诺德、礼来 | 建立疗效预期、医生教育和患者品牌心智。 |
| 国产创新 | 信达、恒瑞、银诺、先为达 | 以机制创新、价格策略和本土渠道速度抢占市场。 |
| 国产替代/类似药 | 华东、九源、石药、齐鲁、联邦 | 通过类似药/仿制药加速价格竞争和准入压力。 |
| MASH 标签教育 | Madrigal、诺和诺德、BI | 影响未来 MASH 诊断、终点、支付和治疗路径认知。 |
| 院内路径 | 内分泌、肝病、减重门诊 | 决定处方启动、风险分层和多学科协作。 |
| 院外路径 | DTP、电商处方药、线上问诊 | 决定可及性、复购和自费患者转化。 |
| 患者长期管理 | 小程序/App、复购提醒、生活方式干预 | 决定依从性、停药管理和长期价值感知。 |

## 8. 中国医保策略

### 总体判断

| 支付领域 | 判断 | 策略含义 |
|---|---|---|
| GLP-1 / T2D | T2D 优先进入医保 | 司美格鲁肽注射液、替尔泊肽、依苏帕格鲁肽 α 等进入医保时主要限定 T2D；院内覆盖和慢病处方连续性是关键。 |
| GLP-1 / 减重 | 自费/商保为主 | 长期体重管理更接近消费医疗+慢病服务混合场景，支付敏感度高，渠道和服务影响转化。 |
| MASH | 支付路径未成熟 | 中国尚无获批 MASH 药物；未来医保需要证明纤维化改善能降低肝硬化、移植和心代谢事件负担。 |
| 商保创新 | 补位基本医保 | 首版商保创新药目录提供新的价格/准入叙事空间。 |

### 医保策略对比

| 策略问题 | GLP-1 / T2D | GLP-1 / 减重 | MASH |
|---|---|---|---|
| 准入目标 | 进入 NRDL、院内覆盖、慢病处方连续性 | 院外可及、商保、体重管理中心套餐 | 先完成标签教育与诊断分层，再谈支付。 |
| 证据重点 | HbA1c、体重、低血糖、心肾获益、真实世界依从性 | 体重降幅、维持、合并症改善、安全性与长期服务 | MASH 缓解、纤维化改善、非侵入检测、长期肝事件。 |
| 价格压力 | 医保谈判和国产替代双重降价 | 患者自付敏感；类似药会压缩溢价 | 初期高价创新药 vs 医保价值证据不足。 |
| BI 动作 | 避免卷入泛 T2D 红海 | 以肝脏和心代谢高危人群建立差异化 | 提前布局经济学模型、筛查路径和真实世界证据。 |

## 9. 对 BI 的战略含义

survodutide 需要提前占位“代谢肝病”。建议定位不是“又一个 GLP-1 减重药”，而是面向肥胖/T2D 合并肝脏风险人群的代谢器官保护方案。

| 优先级 | 战略动作 | 具体内容 |
|---|---|---|
| P0 | 建立疾病框架 | 把 MASH 与肥胖/T2D 高危人群连接，避免被理解成单一减重药竞争。 |
| P1 | 打造筛查路径 | 联合肝病、内分泌、体重管理中心，推动非侵入检测与转诊路径。 |
| P2 | 构建证据叙事 | 从体重、肝脂、ALT/AST 到纤维化指标，形成多终点价值阶梯。 |
| P3 | 设计数字闭环 | 风险测评、复诊提醒、生活方式跟踪、药房复购与 KOL 内容矩阵。 |

## 13. 结论

短期竞争看 GLP-1 红海与支付能力，中期差异化看 MASH/肝脏获益，长期胜负看真实世界管理闭环。

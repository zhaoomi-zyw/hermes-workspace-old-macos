Executive Summary · 三个核心判断

01

竞争升维

驱虫药竞争已从"谁有产品"转向"谁能管理宠主全生命周期"。产品组合 + 渠道触点 + 复购运营 决定未来份额。

02

数字化窗口期

竞品仍在传统渠道竞争。BI 率先用 IT 把驱虫药从 一次性购买 变成 周期性健康管理，是结构性优势。

03

6 个产品机会

驱虫提醒 → 宠主CRM → 兽医助手 → 渠道看板 → 竞品雷达 → 小程序。12 个月可分三阶段落地。

当前增长卡点 · 5 大瓶颈

😴

宠主忘记驱虫

驱虫是低频行为（月/季），宠主容易遗忘。缺乏有效的提醒和复购触发机制。

🐱

室内猫误区

"不出门不用驱虫"观念普遍。需要宠主教育，但触达效率低。

🔄

复购链路断裂

医院→药店→电商，渠道割裂。宠主在哪买、何时买，品牌不可见。

📊

渠道数据不透明

经销层级多，动销/库存不可实时查看。决策依赖滞后报表而非实时数据。

🏥

兽医教育分散

医生需要产品对比、适应症、剂型选择参考，但信息散落在不同渠道，缺乏系统化工具。

IT 产品机会 ①

⏰ Deworming Reminder · 驱虫提醒工具

解决什么

核心痛点：宠主忘记驱虫 → 复购断点 → 品牌流失

关键功能

  • 📅 驱虫日历（月/季度自动提醒）
  • 📱 微信模板消息推送
  • 🛒 一键跳转复购链接
  • 🐕 按宠物体重/品种推荐剂量
  • 📝 驱虫记录时间线
  • 👨‍👩‍👧 多宠管理

产品逻辑

输入：宠主注册 → 宠物档案（种类/体重/年龄）→ 上次驱虫日期
计算：推荐驱虫周期 → 下次驱虫日期
触发：提前 3 天微信提醒 → 驱虫日当天提醒 → 逾期 3 天再提醒
转化：提醒页 → 产品详情 → 电商/医院购买
